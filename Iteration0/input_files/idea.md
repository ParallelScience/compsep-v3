**Title**: Multi-Scale Feature-Decoupling Network for tSZ-CIB Separation and Cluster Extraction

**Description**: We propose a dual-branch U-Net architecture designed to perform simultaneous supervised estimation of the tSZ Compton-y signal and a spatially-varying CIB-contamination mask. To address the 10¹⁵ variance disparity, the network employs a hierarchical loss function that applies a heavy focal weight (10³–10⁴) to cluster-dense regions ($y > 10^{-7}$) while utilizing a secondary branch to explicitly reconstruct the CIB at 857 GHz as a learnable nuisance foreground. The model architecture integrates skip-connections with attention-gating to suppress high-ℓ noise where the SO LAT beam-suppression renders pixel-wise recovery physically impossible. Instead of global spectral loss, we enforce a linear spectral consistency constraint restricted to ℓ > 1000 and high-SNR multipole bins to ensure physical fidelity. The pipeline is validated against the MMF baseline by comparing the cluster selection function—specifically completeness and purity—as a function of mass, utilizing the transfer function T(ℓ) to quantify the mitigation of CIB-induced false detections. By treating CIB contamination as a learned segmentation task, this approach provides a robust, non-generative alternative for recovering integrated-Y in the presence of extreme foreground confusion.
